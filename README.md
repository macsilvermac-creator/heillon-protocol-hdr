# HEILLON Protocol — HDR

Landing page institucional para o protocolo HEILLON, apresentando a Infrastructure de Verificação de Decisões.

## Execução

Abra `index.html` diretamente no navegador ou use um servidor local:

```bash
# Python
python -m http.server 8000

# Node (npx)
npx serve .
```

## Estrutura

- `index.html` — Estrutura principal
- `css/styles.css` — Estilos e variáveis de design
- `js/main.js` — Interações mínimas

## Princípios

- HEILLON é infraestrutura, nunca produto ou SaaS
- Referências a HEILLON no feminino em português
- Tom institucional e minimalista
- Modelo: Intent → Validation → Execution
